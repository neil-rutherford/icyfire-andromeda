Unfollow a user

If you previously followed a user but no longer want to see their content and activity, you can unfollow them.

  1. Log in, if you haven't already.
  2. Go to the user's profile page by clicking on their username or profile picture.
  3. In the left column under the user's profile picture, click the blue button.

If you successfully follow the user, you will see a message that says "You are no longer following [USERNAME]."
