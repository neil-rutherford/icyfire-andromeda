Subscribe to Premium API

The Galatea API has two tiers -- free and premium. The free tier is designed for hobbyists and proofs-of-concept. As a result, it has severe permissions and rate limit restrictions. If you would like to use the Galatea API in a production environment, we highly recommend upgrading to the Premium API Tier.

With the Premium API Tier, clients may increase their rate limit to 3,600 requests per hour (as opposed to 60 requests per hour in the free tier) and apply for escalated permissions.

Before beginning your subscription, you should have already created an app. Go to the "Start your Premium API subscription" page. Clicking the green will begin the payment process.

You may cancel your subscription at any time.
