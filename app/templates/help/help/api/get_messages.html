Get a list of your messages

Returns a list of your messages.

Endpoint /_api/get/messages
Method POST
Form-encoded POST input { 'consumer_key': [CONSUMER_KEY], 'consumer_secret': [CONSUMER_SECRET] }
On success
  • 200 status code
  • JSON array of objects
  • [ { 'id': Primary key (int), 'from_user_id': Sender user ID (int), 'to_user_id': Recipient user ID (int), 'subject': Subject line (str), 'sent_timestamp': When was the message sent, in UTC? (datetime), 'seen_timestamp': When was the message seen, in UTC? (datetime) }, { ... } ]
Example response [{ 'id': 4456, 'from_user_id': 56, 'to_user_id': 1, 'subject': 'I loved your article!', 'sent_timestamp': 2021-08-24T11:04:51.431223Z, 'seen_timestamp': 2021-08-24T11:06:51.123456Z }, { 'id': 5632, 'from_user_id': 1423, 'to_user_id': 1, 'subject': 'Wanna come over for dinner?', 'sent_timestamp': 2021-08-24T11:04:51.431223Z, 'seen_timestamp': 2021-08-24T11:06:51.123456Z }]
Error messages
  • 403 (Client not found): Consumer key is incorrect.
  • 403 (Incorrect credentials): Consumer secret is incorrect.
  • 403 (API access has been restricted): Your credentials are correct, but your app is either not approved or on a blacklist.
  • 403 (Insufficient permissions): You don't have permission to access that resource. (e.g. Trying to access users without the "users" permission.)
  • 429 (Too many requests): You have exceeded your rate limit.
