Create an app

Before you may begin using the Galatea API, you must first register an app. To do this, go to the "Apply for API permissions" page and fill out the form. The information you provide will help us guide you through the process. If we find that you have been untruthful in your application, we reserve the right to adjust your permissions as we see fit.

  • Company name: If you are an individual, please write "Individual". Otherwise, please enter your corporate entity's legal name.
  • Contact name: If you are an individual, please write your full legal name. Otherwise, please enter a point of contact for your company.
  • POC phone number: Please enter your/your representative's 10-digit U.S. or Canadian phone number. Foreign phone numbers are not accepted at this time.
  • POC email: Please enter your/your representative's email address. It can be the same as the email address you used to register for an account on Galatea.

Standard API clients are restricted to the following standard permissions:

  • Articles: It allows you to view existing articles, as well as create and delete your own (if you have an active Patriot Tier subscription).
  • Comments: It allows you to view existing comments for a given article, as well as create and delete your own.
  • Users: It allows you to view users' information, as well as create and delete user accounts.
  • Messages: It allows you to create, retrieve, and delete messages.

Premium API clients may apply for the following escalated permissions:

  • Logs: It allows you to view detailed activity logs about website visitors.
  • Shares: It allows you to view detailed logs about social media propagation.
  • Connections: It allows you to view relationships between users, as well as the nature of their relationship.
  • API: It allows you to review and approve API client applications.

Once you have chosen the permissions you plan to use and provided a justification, move on to the next section and tell the reviewers a little more about your app:

  • Use case: How do you plan to use the API?
  • Developer platform: What platform are you developing your app for?
  • Monetization model: If you plan to make money from the Galatea API, please explain here. Otherwise, enter "N/A".

After you submit your application, your app will go into a review stage and you will receive a response within 3 business days. You may check on the status of your app here. If reviewers have any questions, they will communicate with you via the email you provided. If you have applied for escalated permissions, subscribe to the Premium API Tier as soon as possible so that your application is not hindered.
