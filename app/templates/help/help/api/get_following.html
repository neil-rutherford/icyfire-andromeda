Get a list of who a user is following

Returns a list of users who a given user follows. Takes user_id (user primary key, as an integer) as an argument.

Endpoint /_api/get/following/[USER_ID]
Method POST
Form-encoded POST input { 'consumer_key': [CONSUMER_KEY], 'consumer_secret': [CONSUMER_SECRET] }
On success
  • 200 status code
  • JSON array of objects
  • [ { 'id': Primary key (int), 'username': Username (str), 'status': User's status (str), 'is_patriot': Is the user subscribed to Patriot Tier? (bool) }, { ... } ]
Example response [{ 'id': 4, 'username': 'real_eric_cartman', 'status': 'Screw you guys, I'm going home!', 'is_patriot': true }, { 'id': 5, 'username': 'mr_mackey', 'status': null, 'is_patriot': false }]
Error messages
  • 403 (Client not found): Consumer key is incorrect.
  • 403 (Incorrect credentials): Consumer secret is incorrect.
  • 403 (API access has been restricted): Your credentials are correct, but your app is either not approved or on a blacklist.
  • 403 (Insufficient permissions): You don't have permission to access that resource. (e.g. Trying to access users without the "users" permission.)
  • 404: User not found.
  • 429 (Too many requests): You have exceeded your rate limit.
